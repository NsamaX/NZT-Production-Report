# NZT Production Report

Internal tool for managing and reporting daily production output at Nozomi (NZT) Company. Supports role-based access, historical tracking, and data export.

## Features

- **Role-Based Access**: Each user role has tailored access to features relevant to their responsibilities.
- **Production Reporting**: Records and reports daily production quantities per model and production line.
- **Historical Analysis**: View production history on a daily, monthly, and yearly basis with chart visualization.
- **PDF Report Generation**: Export production summaries as professional PDF reports.
- **Excel Data Export**: Export production data to Excel for further analysis.
